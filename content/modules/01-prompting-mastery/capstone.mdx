## The Project

A personal library of 5 reusable prompt templates for the tasks you actually do. Not hypothetical prompts — real prompts for real work, tested against Claude, and refined until they produce results you'd actually use. The Prompt Playbook is the first artifact of your AI-powered workflow and you'll reference it throughout the rest of the masterclass.

## What You'll Build

- 5 prompt templates covering 5 different use cases from your own work or life (examples: email drafting, research summaries, decision analysis, content creation, code review explanations)
- Each template follows the 5-part anatomy: Role, Context, Task, Constraints, Output Format — with placeholders in [BRACKETS] for the parts that change each time
- A tested result for each template: paste the template into Claude.ai, fill in the brackets, and verify the output is genuinely useful
- A one-sentence note per template: what makes this prompt effective (the "why")

## Skills You'll Use

- Lesson 2 (Anatomy): The 5-part structure is the foundation of each template
- Lesson 3 (Specificity): Each template must be specific enough that Claude's first response is usable
- Lesson 5 (Iteration): Test each template, refine it at least once
- Lesson 6 (Personas): At least 2 templates should use a Role component
- Lesson 7 (Structured Output): At least 2 templates should specify an output format (table, bullets, JSON)

## Getting Started

1. List 5 tasks you do repeatedly that could benefit from a consistent Claude prompt (e.g., summarizing meeting notes, drafting follow-up emails, analyzing a decision)
2. For each task, write a draft prompt using the 5-part anatomy template from Lesson 2
3. Test each prompt in Claude.ai, note what's missing, and refine once

## Success Criteria

- [ ] 5 prompt templates exist (written down — in a doc, notes app, or plain text file)
- [ ] Each template has all 5 anatomy components (Role, Context, Task, Constraints, Output Format)
- [ ] Each template has been tested in Claude.ai and produced a result you would actually use
- [ ] At least 2 templates use a persona (Role component)
- [ ] At least 2 templates specify a structured output format
- [ ] You have a one-sentence "why this works" note for each template

## Stretch Goals

- Expand to 10 templates covering more of your recurring tasks
- Add a "when to use this prompt" note to each template
- Share one template in a format you could give to someone else who does the same work
